# AWS IAM Security Auditor

## Overview

AWS IAM Security Auditor is a cloud security project designed to identify common Identity and Access Management (IAM) security issues within AWS environments.

## Objectives

* Identify overly permissive IAM users
* Detect administrative privileges
* Review user and role permissions
* Generate basic security recommendations

## Technologies

* Python
* AWS IAM
* JSON
* GitHub
